# Agent Pack — INVARIANTS

Критичные вещи, которые легко нарушить случайно. Полный свод правил — в `rules.md`.

---

## Изоляция данных

- Данные одного тенанта **никогда** не должны быть доступны другому.
- Любые операции с чужим идентификатором тенанта → 403.
- В MVP: идентификатор тенанта фиксирован для инсталляции (из seed, не из запроса).
- Создание тенантов через публичный API **отключено намеренно**.

## Публичный контракт

- Публичный контракт — только то, что доступно через **API Gateway**.
- Внутренние эндпоинты сервисов — не часть контракта для клиентов.
- Изменение публичного контракта требует явного согласования.

## AI-специфично

- AI не может напрямую изменять бизнес-данные — только предлагать изменения.
- Промпты — это конфигурация, не хардкод в коде.
- Каждый AI-ответ должен быть трассируем до конкретного события.
