# Agent Pack — START

Этот набор файлов предназначен для ИИ-агента, работающего над разработкой проекта.

## Политика чтения (чтобы держать контекст "лёгким")

**По умолчанию агент читает только:**

1. `agent/*` (этот набор)
2. `phases/*` (по фазам: что делать / что сделано / тесты)

**Все остальные документы** (включая `sources-of-truth/*`) — **только по запросу (retrieval)**,
когда это реально нужно для конкретной задачи.

## Что читать дальше

1. `agent/INVARIANTS.md` — ограничения и "запреты" (то, что нельзя нарушать)
2. `agent/PUBLIC_API.md` — что считается публичным контрактом и как с ним работать
3. `agent/TESTING.md` — как проверять изменения быстро и безопасно

Если задача привязана к конкретной фазе — переходи в `phases/phase-X/WHAT_TO_DO.md`.

## Принцип работы с фазами

Каждая фаза содержит три файла:
- `WHAT_TO_DO.md` — что нужно сделать (задачи)
- `DONE.md` — что уже сделано (отчёт)
- `TESTS.md` — как проверить результат

Агент работает с текущей активной фазой. Завершённые фазы — только для справки.
